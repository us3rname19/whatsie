## How to install

### Linux
### Ubuntu, Debian 8+ (amd64.deb package):


```bash

wget https://github.com/us3rname19/whatsie/releases/download/v2.1.0/whatsie-2.1.0-linux-amd64.deb

sudo dpkg -i whatsie-2.1.0-linux-amd64.deb

whatsie

```
